## 📋 Description

Ce projet contient l'implémentation complète des 6 exercices du TD sur les **Design Patterns** réalisé dans le cadre de mes études à l'**EMSI MARRAKECH** (Année universitaire 2025-2026).

Chaque exercice illustre l'utilisation d'un ou plusieurs patterns de conception avec des exemples concrets en Java.
